# chip-8-emulator
A CHIP-8 emulator (see [CHIP-8 Wikipedia](https://en.wikipedia.org/wiki/CHIP-8)) written in C++20.

## Build instructions
```shell
./helper/build.sh
./build/bin/CHIP8_EMULATOR example_rom.c8
```
